# ChekerUsernameTelegram
Программа для массовой проверки доступности юзернеймов в Telegram. Позволяет быстро находить свободные юзернеймы по заданному шаблону и диапазону цифр.
# Требования 
· Python 3.6 или выше\
· Активный аккаунт Telegram\
· API данные от my.telegram.org
